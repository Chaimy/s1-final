h1. Bricks

h2. Basic Usage

Create a Table object

<pre>
table = Bricks::Table.new
</pre>

Add a row

<pre>
  table = Bricks::Table.new
  table.columns             # => []
  table.rows                # => []
  table.add_row [1, 2, 3]
  table.columns             # => [[1], [2], [3]]
  table.rows                # => [[1, 2, 3]]
</pre>
